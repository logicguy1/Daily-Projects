# Encode Morse | Day 2

Create a function that takes a string as an argument and returns the Morse code equivalent.

## Examples:
```py
encode_morse("EDABBIT CHALLENGE") ➞ ". -.. .- -... -... .. -   -.-. .... .- .-.. .-.. . -. --. ."

encode_morse("HELP ME !") ➞ ".... . .-.. .--.   -- .   -.-.--"
```
